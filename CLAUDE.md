## Git Operation Rules

**Every code change must be committed and pushed to GitHub.**

After completing any code change:

```bash
git add <changed files>
git commit -m "concise description of change"
git push origin <current branch>
```

- Commit frequently with meaningful messages
- Never leave uncommitted changes at the end of a task
- Always push after committing — local-only commits are not acceptable
- Use the current branch (check with `git branch --show-current` if unsure)
- If the remote branch does not exist yet, use `git push -u origin <branch>`
